# Controle Financeiro Doméstico

Um sistema simples para controle financeiro doméstico, com front-end em HTML/CSS/JS e back-end em PHP puro, utilizando MySQL como banco de dados.

## Estrutura do Projeto

- `frontend/`: Arquivos do front-end (HTML, CSS, JS)
- `backend/`: Arquivos do back-end (PHP)
- `database/`: Scripts SQL para o banco de dados

## Configuração

1. Clone o repositório.
2. Execute o script SQL em `database/schema.sql` para criar o banco de dados e tabelas.
3. Configure as credenciais do banco em `backend/config.php`.
4. Abra `frontend/index.html` no navegador (pode precisar de um servidor local para o PHP).

## Funcionalidades

- Adicionar receitas e despesas
- Listar transações
- Categorias para organizar
